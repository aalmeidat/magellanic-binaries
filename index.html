The Small Magellanic Cloud (SMC)

To determine the true binary fraction of the SMC, we first applied a rigorous astrometric cut using Gaia DR3 proper motions to remove Milky Way foreground interlopers. We then utilized the Bayesian sampler The Joker on multi-epoch radial velocities from APOGEE.

Figure 1: Gaia DR3 Proper Motion selection for SMC targets. The elliptical boundary isolates the dense core of the galaxy's proper motion.

The Binary Fraction Results

Using a maximum likelihood estimator on the posterior distributions of 1,466 pure SMC members (SNR > 30), we estimate the close binary fraction (P < 1000 days, M2 > 0.1 M_sun) to be roughly 4.10%.
